# <h1>Active Directory Home Lab</h1>

<h2>Description</h2>
The goal of the lab is to simulate a small business network environment where centralized authentication and management can be practiced. By creating this setup, you can learn how domains work, how users and computers are managed, and how administrators apply security policies across systems.
<br />


<h2>Languages and Utilities Used</h2>

- <b>PowerShell</b> 
- <b>Diskpart</b>

<h2>Environments Used </h2>

- <b>Windows 10</b> (21H2)
